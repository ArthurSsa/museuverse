## 🎯 Sobre o projeto

O MuseuVerse é uma experiência 3D interativa que recria parte do acervo do **Museu Nacional do Rio de Janeiro**, destruído pelo incêndio de setembro de 2018. O visitante explora um saguão virtual em primeira pessoa, interage com peças icônicas perdidas e, ao concluir a visita, recebe um certificado on-chain (NFT) como prova de participação na preservação dessa memória.
