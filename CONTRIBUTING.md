# Como contribuir

## Branch
- Trabalhamos direto na branch `main` (projeto curto, 11 dias).

## Mensagens de commit
Formato: `[area] descrição em minúsculas`

Áreas:
- `[player]` — controle de personagem
- `[ui]` — interface de usuário
- `[interaction]` — sistema de interação
- `[audio]` — sistema de áudio
- `[progress]` — sistema de progresso
- `[web3]` — integração blockchain
- `[contract]` — smart contract
- `[model]` — assets 3D
- `[docs]` — documentação
- `[chore]` — configuração, gitignore, etc.

Exemplos:
- `[player] adiciona controle de camera em primeira pessoa`
- `[ui] cria painel curatorial das pecas`
- `[contract] deploy do erc721 na polygon amoy`

## Pull / Push
1. **Fetch origin** antes de começar a trabalhar (botão no topo do GitHub Desktop)
2. Trabalhe localmente
3. Commit no GitHub Desktop com mensagem padrão
4. **Push origin**

## Conflitos
Se der erro de conflito ao fazer push:
1. Faça **Fetch origin** primeiro
2. O GitHub Desktop avisa se há conflitos e oferece interface pra resolver
3. Resolva, commit, push

Em caso de dúvida, chame no canal da equipe.
